Introdução¶

A pandemia da doena do novo coronavrus (COVID-19) causada pelo vrus SARS-CoV-2 tem causado grande impacto em todo o mundo, inclusive no Brasil. Muitos dados foram coletados e analisados para entender melhor a disseminao e o impacto dessa doena. Este documento usa dados disponveis no conjunto de dados em: https://brasil.io/dataset/covid19/files/ (https://data.brasil.io/dataset/covid19/caso_full.csv.gz)

Objetivo¶

O objetivo deste estudo analisar os dados do COVID-19 disponveis nos conjuntos de dados acima e documentar as principais lies aprendidas. Examine informaes como nmero de casos, bitos, taxas de letalidade, distribuio geogrfica e outros aspectos relevantes para entender a evoluo da pandemia no Brasil. metodologia

Metodologia¶

Coleta de dados: Baixe o conjunto de dados disponvel em https://data.brasil.io/dataset/covid19/caso_full.csv.gz.
Os dados incluem os seguintes detalhes sobre os casos de COVID-19 no Brasil: Data, regio, municpio, idade, sexo, estado de sade, etc.

  • Pr-processamento de dados:
    Execute as etapas de pr-processamento de dados para que os dados estejam em um formato adequado para anlise. Isso inclui limpar dados ausentes, transformar variveis e filtrar informaes suprfluas.

  • Anlise exploratria de dados:
    Aplique tcnicas de anlise exploratria de dados para obter insights iniciais. Isso pode incluir a criao de grficos, tabelas e medidas estatsticas para descrever e resumir dados.

  • Anlise Adicional:
    Com base nos resultados de nossa anlise exploratria, conduziremos anlises adicionais para responder a perguntas especficas sobre a disseminao e o impacto do COVID-19 no Brasil. Isso pode incluir a identificao de tendncias ao longo do tempo, anlise da distribuio geogrfica dos casos, correlaes entre variveis e outras anlises relevantes.

# 4. Grfico apresenta o percentual de pessoas infectadas em relao  populao estimada
import pandas as pd
import matplotlib.pyplot as plt

# Agrupar por estado, somando os casos confirmados e a populao estimada
df_grouped = df.groupby('state').agg({'new_confirmed': 'sum', 'estimated_population': 'first'}).reset_index()

# Calcular o percentual de pessoas infectadas em relao  populao estimada
df_grouped['percent_infected'] = (df_grouped['new_confirmed'] / df_grouped['estimated_population']) * 100

# Remover linhas com valores NaN na coluna "percent_infected"
df_grouped = df_grouped.dropna(subset=['percent_infected'])

# Ordenar o DataFrame pelo percentual de pessoas infectadas em ordem decrescente
df_sorted = df_grouped.sort_values(by='percent_infected', ascending=False)

# Criar uma figura com dois subplots (um abaixo do outro)
fig, axes = plt.subplots(2, 1, figsize=(10, 12))

# Plotagem de barras verticais
axes[0].bar(df_sorted['state'], df_sorted['percent_infected'], color='red')
axes[0].set_xlabel('Estado')
axes[0].set_ylabel('Percentual de Pessoas Infectadas')
axes[0].set_title('Percentual de Pessoas Infectadas por Estado (Barras Verticais)')

# Plotagem de barras horizontais
axes[1].barh(df_sorted['state'], df_sorted['percent_infected'], color='blue')
axes[1].set_xlabel('Percentual de Pessoas Infectadas')
axes[1].set_ylabel('Estado')
axes[1].set_title('Percentual de Pessoas Infectadas por Estado (Barras Horizontais)')

plt.tight_layout()  # Ajustar o espaamento entre os subplots
plt.show()
