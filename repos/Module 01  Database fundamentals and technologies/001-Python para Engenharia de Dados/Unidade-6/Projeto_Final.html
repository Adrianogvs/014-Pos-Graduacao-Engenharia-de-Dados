Projeto Final - Análise de Dados COVID-19¶

Introdução¶

A pandemia da doença do novo coronavírus (COVID-19) causada pelo vírus SARS-CoV-2 tem causado grande impacto em todo o mundo, inclusive no Brasil. Muitos dados foram coletados e analisados para entender melhor a disseminação e o impacto dessa doença. Este documento usa dados disponíveis no conjunto de dados em: https://brasil.io/dataset/covid19/files/ (https://data.brasil.io/dataset/covid19/caso_full.csv.gz)

Objetivo¶

O objetivo deste estudo é analisar os dados do COVID-19 disponíveis nos conjuntos de dados acima e documentar as principais lições aprendidas. Examine informações como número de casos, óbitos, taxas de letalidade, distribuição geográfica e outros aspectos relevantes para entender a evolução da pandemia no Brasil. metodologia

Metodologia¶

Coleta de dados: Baixe o conjunto de dados disponível em https://data.brasil.io/dataset/covid19/caso_full.csv.gz.
Os dados incluem os seguintes detalhes sobre os casos de COVID-19 no Brasil: Data, região, município, idade, sexo, estado de saúde, etc.

  • Pré-processamento de dados:
    Execute as etapas de pré-processamento de dados para que os dados estejam em um formato adequado para análise. Isso inclui limpar dados ausentes, transformar variáveis e filtrar informações supérfluas.

  • Análise exploratória de dados:
    Aplique técnicas de análise exploratória de dados para obter insights iniciais. Isso pode incluir a criação de gráficos, tabelas e medidas estatísticas para descrever e resumir dados.

  • Análise Adicional:
    Com base nos resultados de nossa análise exploratória, conduziremos análises adicionais para responder a perguntas específicas sobre a disseminação e o impacto do COVID-19 no Brasil. Isso pode incluir a identificação de tendências ao longo do tempo, análise da distribuição geográfica dos casos, correlações entre variáveis e outras análises relevantes.

# 4. Gráfico apresenta o percentual de pessoas infectadas em relação à população estimada
import pandas as pd
import matplotlib.pyplot as plt

# Agrupar por estado, somando os casos confirmados e a população estimada
df_grouped = df.groupby('state').agg({'new_confirmed': 'sum', 'estimated_population': 'first'}).reset_index()

# Calcular o percentual de pessoas infectadas em relação à população estimada
df_grouped['percent_infected'] = (df_grouped['new_confirmed'] / df_grouped['estimated_population']) * 100

# Remover linhas com valores NaN na coluna "percent_infected"
df_grouped = df_grouped.dropna(subset=['percent_infected'])

# Ordenar o DataFrame pelo percentual de pessoas infectadas em ordem decrescente
df_sorted = df_grouped.sort_values(by='percent_infected', ascending=False)

# Criar uma figura com dois subplots (um abaixo do outro)
fig, axes = plt.subplots(2, 1, figsize=(10, 12))

# Plotagem de barras verticais
axes[0].bar(df_sorted['state'], df_sorted['percent_infected'], color='red')
axes[0].set_xlabel('Estado')
axes[0].set_ylabel('Percentual de Pessoas Infectadas')
axes[0].set_title('Percentual de Pessoas Infectadas por Estado (Barras Verticais)')

# Plotagem de barras horizontais
axes[1].barh(df_sorted['state'], df_sorted['percent_infected'], color='blue')
axes[1].set_xlabel('Percentual de Pessoas Infectadas')
axes[1].set_ylabel('Estado')
axes[1].set_title('Percentual de Pessoas Infectadas por Estado (Barras Horizontais)')

plt.tight_layout()  # Ajustar o espaçamento entre os subplots
plt.show()
